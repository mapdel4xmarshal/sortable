HOW TO RUN ON MAC
-----------------
* Clone from git

* Run terminal

* Navigate to the cloned directory from terminal

* Run “python sortable.py”



HOW TO RUN ON WINDOWS
---------------------
* Clone from git

* Run cmd prompt

* Run "path\to\python\installation\python.exe path\to\cloned\directory\sortable.py

NOTE : 

--------------
In order to ensure backward compatibility with python 2.7, the “io” module was used to handle encoding. As a result of this, the script takes around 2 minutes to successfully execute.
